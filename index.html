Statistical Simulation and Inference in the Browser

StatSim is a free probabilistic simulation web app. Multiple simulation methods and over 20 built-in distributions make it possible to create complex models and implement Bayesian inference in the browser. Server-side processing coming soon...

Open web app or Download

Features

Deterministic Programs

Create and share regular programs just in your browser. From simple calculators to complex environmental models.

Stochastic Modeling

Use 24 probability distributions to build stochastic models. Analyze results with charts and multiple statistics.

Bayesian Inference

Estimate the posterior distributions of parameters using the Markov Chain Monte Carlo and Hamiltonian MC algorithms.

Neural Networks

Create and train feed-forward neural networks using the simple and intuitive graphical interface.

Create new and expore existing models

Build models in editor mode using different types of blocks, try multiple simulation methods online (Rejection Sampling, MCMC, HMC and other)

Keep only what matters

Use and share your models in preview mode to hide inner logic, simulation details and keep only what matters - input and output

Analyze results

Various charts and descriptive statistics make it easier to extract insights from data and samples

Compile

Compile your models into system binaries using external server to use them as regular programs and get additional stats and speed improvements
